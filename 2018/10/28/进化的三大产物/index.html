进化的三大产物

发表于 2018-10-28 | 分类于 生命科学
字数统计: 1,415 | 阅读时长 ≈ 5

三大产物

进化的过程中有 三大产物：

  1. 适应器（adaptations）
  2. 适应器的副产物（by-products）
  3. 随机影响（random effects） / 噪音（noise）

适应器

适应器可以定义为一种 通过自然选择形成的，遗传性的，能够稳定发展起来的特性。而在进化的过程当中，它能够帮助有机体解决生存和繁衍问题。

定义的 核心部分：

  1. 适应器 必须拥有相应的基因，换而言之就是适应器 拥有某一种遗传基础，多数的适应器并不能够被追溯到某一个具体的基因片段上面，而是许多基因的合作产物
  2. 适应器必须处于 “正常”环境 中的种族成员身上才能够恰当的发展起来，换而言之，适应器一定是在有机体生命史中恰当的阶段以一种 完整无损的形式 出现，并最终成为种族的大多数或者全体成员所 广泛具有的特性。不可否认，有一些机制就只能够在某一性别成员身上存在，或者存在于特定的子群体当中。但是 绝大多数 的适应器都是特定种族的典型特征
  3. 适应器总是能够适当的发展起来的，但是这不意味着所有的适应器就必须在出生的时候就显现
  4. 适应器是由选择过程所塑造出来的，选择会作用于每一代，将那些无助于繁衍的特征过滤掉，而那些有助于繁衍的特征保留下来

适应器的 功能 是指它进化出来为了解决某种适应性的问题，换而言之，它如何才能够 有利于生存和繁衍。
评估一个适应器的功能的标准通常包括：

  1. 有效性（以高效的方式来解决问题）
  2. 经济性（以解决成本的方式来解决问题）
  3. 精确性（所有的组成部分都专门用于实现特殊的目的）
  4. 可靠性（在相应的环境下能够稳定的表现出来）

每种适应器都会有自身的进化阶段，一开始是 突变（mutation), 即某一个DNA片段的复制错误，而这种事情 只发生在单个个体 上， 从统计上来讲，大部分的突变都会阻碍个体的生存和繁衍，但是有些偶然的突变能够给个体的生存和繁衍带来益处。如果一种突变给有机体带来的 繁殖优势 超过了群体中的其他成员，拿它就能够生育更多的后代。这样一来 下一代中将会有更多的个体拥有这种基因突变所表现的特征。如果这种基因 一直拥有繁殖优势，那么经历许多世代之后，这种突变基因将会 扩散 到某个群体身上，直至每个成员都拥有了这种基因。

共识：适应器是选择和进化过程中的主要产物。

包括人在内的物种都是有适应器组成的，其中有一些适应器是感觉器官：眼睛，耳朵，鼻子等等。而有些适应器能够让人在环境中自由活动：如直立的骨架。

适应进化的环境

适应进化的环境（environment of evolutionary adaptedness, EEA） 是指在 适应器的进化阶段中促使适应器形成的选择压力的统计学总结。 每种适应器的EEA是指在长久的进化历史中塑造出该适应期的那些选择力量，或者成为适应性问题。需要主要的是：EEA并不是指向特定的时间或地点，而是指塑造适应器的选择压力。所以，每一种适应器都有其独一无二的EEA。所谓 适应器的进化阶段（period of evolution），指的是 适应器从一开始一点一点发展起来，最终成为种族内普遍拥有的设计特征的整个时间跨度。

副产品

适应器不是进化的唯一产物，附带的产物即是 副产物。副产物 既不能够解决适应性问题，也不能够解决功能性的设计特征。副产物是由那些具备功能的设计特征“附带”而来的，他们只是碰巧和适应器结合在一起，比如手灯泡的设计出来是为了照明，而发热是这一个目的的副产物。肚脐眼就是一个典型的副产品，没有任何的证据证明肚脐眼能够对人的生存和繁衍带来任何的帮助，但是脐带作为一种适应器能够为胎盘提供必需的营养，而脐带就是胎盘的副产品。

随机影响

随机影响（噪音）可能由 基因突变，环境中突发的剧烈变化和个体在发展的过程中遭遇的意外等 情况引起的。有些情况之下噪音会对有机体的正常功能导致损害，比如说将咖啡倒入到电脑里面或者将沙子撒入到机器里面；但是有些情况之下就没有影响，比如说灯泡的表面不够光滑不影响灯泡本身的照明功能。
噪音和适应器的副产品是不同的，因为它和适应器设计特征没有任何的关联，是独立形成的。
