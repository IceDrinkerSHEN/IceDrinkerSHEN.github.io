一万小时定律

在《异类》中，作者通过引用调查和科学实验告诉人们：

人们眼中的天才之所以卓越非凡，并非天资超人一等，而是付出了持续不断的努力。只要经过1万小时的训练，任何人都能从平凡变成超凡。

这也被称之为叫做一万小时定律。

一万小时定律虽然反驳了天赋论，但是也迎来了心理学界的质疑：

  1. 为什么是一万个小时，九千九百九十九不行吗？如果不行，那么第一万个小时的训练到底最整体起到了什么样的作用？可以，那么如此往复，九千九百九十八个小时呢？
  2. 不同专业领域的技能习得时间和练习时间是不一样的。乐器，体育运动，国际象棋，钢琴等等，所需的技能和时间并不相同。
  3. 练习的成果并不与时间呈正相关。并不是投入时间就会有产出。有针对性的练习和调整，不断调整练习策略，弥补漏洞的「刻意练习」方式，和没有方法的机械练习有本质不同。

天才是训练的结果

训练的越早，某些能力便越容易被培养。比如说孩子更加容易被培养出绝对音域。

训练过程中讲究方法，有出色的导师指引进行有目标的培训。莫扎特的父亲就是小提琴家。

通过专注的训练来促使大脑改变，使他们能做到超出正常情况的事情。也就是通过专注模式来进行训练，从而加强自己的神经元联结。

为什么多数人没有被训练成天才

训练可以分为两种：

  1. 有效训练
  2. 无效训练

大多数人没有被训练成为天才是因为他们的训练多为无效训练，训练方法不对。

而他们的训练方法之所以错误，是因为训练者有以下三个误区：

  1. 他们持有了基因决定论的观点，认为后天的努力无法改变什么或者只能带来微弱的改变
    1. 但事实上任何人都能够通过后天的正确的训练来改变和提高自己
  2. 只要练习的时间足够长，能力就能够变强
    1. 训练时间的长短和能力的提升并没有显著的影响
    2. 没有正确的方法指导的反复练习并不能够提升自己，反倒是会让人止步不前和能力逐渐下降
  3. 只要足够努力刻苦，就能够提高成绩
    1. 除非你有专门的恰当的方法，否则加倍努力带来的进步并不大

进行有效的训练

有效的训练是一种有目的的训练：

  1. 确定目标并且不断的改进
  2. 在进行训练的时候必须专注
  3. 训练要有着及时反馈
  4. 必须不断的跳出“舒适区”来进行训练

最强大的训练方法是刻意练习：刻意练习的目的在于建立一个更强大的心里表征来思考问题，这也是迄今为止最为强大和有效的训练方式。

刻意练习有着两个标准：

  1. 所训练的领域需要是合理发展的领域，有着一套成熟且完整的评价标准和高效的方法
    1. 对于效绩的测量，具有着客观性
    2. 具有竞争性，能够激发从业者的冬季来训练和提高
    3. 存在着已经被实践过的，具备科学性的训练方法
    4. 在该领域内，最杰出的从业者已经达到了一定程度的表现水平，并且有着一套行之有效的训练方法
  2. 需要有一个能够给自己布置训练作业和及时反馈的优秀的导师
    1. 寻找该领域的成功者的帮助，让他们帮忙指明学习路线和训练项目，并且及时的对训练结果做出反馈
    2. 对成功者的模仿是提升能力的一种重要的方法

刻意练习也存在着四个原则：

  1. 明确高绩效的目标，也就是哪些方面需要提升
  2. 尽可能的去寻找这个领域的专家，成功者，高手或者经典的书籍，以此来和自己进行不断的对比
  3. 研究那些杰出的人，去研究他们杰出背后可能的原因是什么
  4. 不断的投入时间和精力来进行训练

刻意练习发生在舒适区之外。Noel Tichy 将学习分成了三个区域：

  1. 舒适区：已经完全掌握的区域，没有学习难度的知识或者习以为常的事务。
  2. 学习区：还没有完全掌握的区域，对你来说有一定挑战，感到不适，但不至于太难受。
  3. 恐慌区：超出自己能力范围太多的事务或知识，心理感觉会严重不适，可能导致崩溃以致放弃学习。

刻意练习的一个关键是持续不断地去尝试那些刚好超出当前能力范围的事物。

最重要的是，刻意练习能形成有效的心理表征。心理表征是一种我们大脑正在思考的某个物体、某个观点、某些信息或者其他任何事物相对应的心理结构，或具体或抽象。简单来说，就是「物体概念在人们脑海中塑造的意义」。

心理表征的作用：

  • 快速找到规律，进行有意义的记忆。
  • 快速处理大量的信息、组织信息。
  • 精准预测未来，在瞬间做出更快更准确的决策。
  • 制定计划，选择最佳路径。
  • 有助于高效学习，可以敏锐地察觉到自己的错误。

技能和心理表征之间的关系是一种良性循环，你的技能越娴熟，创建的心理表征越好，而心理表征越好，就越能有效地提升技能。
