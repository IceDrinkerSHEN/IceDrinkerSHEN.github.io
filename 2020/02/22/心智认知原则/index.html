心智认知原则

发表于 2020-02-22 | 分类于 原则
字数统计: 3,294 | 阅读时长 ≈ 11

  1. 建立具体的目标

    1. 目标的建立要满足Smart法则
      1. 目标必须是具体的（Specific）
        　　2. 目标必须是可以衡量的（Measurable）
        　　3. 目标必须是可以达到的（Attainable）
        　　4. 目标必须和其他目标具有相关性（Relevant）
        　　5. 目标必须具有明确的截止期限（Time-based）
    2. 将大的目标分解成小目标
      1. 找出这一项任务的终极目的
      2. 分解小目标不仅仅是开始点，结束点和中间的一条直线
      3. 小目标要构成一个前往大目标的路线
        1. 一到五项可以产生反馈的小结果
      4. 某一个小目标也都要满足Smart法则
      5. 如果目标不是一个为开拓的领域，那么去请教这个领域的专家或者已经成功过的人
    3. 建立每一个时间段的目标
      1. 将小目标分到各个时间段里面
      2. 时间段为年/月/周
      3. 年目标由月目标构成，月目标由周目标构成

  2. 给任务进行优先级排序

    1. 采用艾森豪威矩阵来进行优先级排序

      1. 重要并且紧急的任务要立即处理
      2. 重要但是不紧急的任务决定什么时候处理
      3. 紧急但是不重要的任务如果能够交给别人来做的尽量交给别人来做
      4. 不重要也不紧急的任务可以稍后再做

    2. 优先级规划需要遵守三个法则

      1. 优先级不能够被创造或者摧毁，只能够从一项任务转移到另一项任务。如果一项任务的优先级上升了，另一项任务的优先级也就随之下降了
      2. 优先级可以进行重新排序。排序时间离现在越久的，参考价值越低
      3. 不要想着同一时间做多件事情，这和什么都不做是一样的

  3. 创建代办清单

    1. 创建一张快捷清单，存放最多五项当前的任务，如果要添加一项新的任务，要么完成一项已有的任务，要么删除一项已有的任务
    2. 创建一张杂草清单，将不重要的无法拒绝的并且不会马上做的任务放置到这个清单中，任务需要包括[目标，利益关系人，日期]等内容。列入该清单时常越长的任务需要越先离开，否则该任务没有必要被完成
    3. 杂草清单没有数量限制，也不需要优先级排序

  4. 减少待办任务

    1. 任务不是越多越好，而是产出的价值越高越好
    2. 创建快捷清单，列出最重要的五项任务，并且按照艾森豪威矩阵来进行优先级排序
    3. 制定本周目标，思考本周要完成的任务是什么，产出的结果是什么
      1. 列出当前和未来的全部任务，而不仅仅是小任务
      2. 挑出最重要的一项任务，将其写在便利贴上，并且贴在显眼的地方
    4. 制定杂草清单，将不重要不紧急但是无法被删除的任务放置到这个清单当中
      1. 每周都要进行割草，保证杂草不会越来越多，和保证杂草清单的可靠性

  5. 为自己的时间区分单核时段和全景时段

    1. 在单核时段中只能够专注于处理快捷清单中的某一项任务
      1. 只用来专心处理某一个任务，不在多任务之间进行切换
        1. 不断的切换任务会导致效率降低
        2. 不断的切换任务会导致更高的出错率
        3. 不断的切换任务会导致大脑中的压力激素皮质醇分泌增加
          1. 会提高人的攻击性
          2. 会降低人的情商和交流能力
        4. 不断的切换任务会消耗人的精力，导致注意力溃散
      2. 不关心之前完成了的任务或者还有哪些没有完成的任务
      3. 至少30分钟的时间用来进行单核处理
    2. 在全景时段里面可以查看自己的任务进度
      1. 将已经完成了的任务从自己的快捷清单里面划掉
      2. 查看自己还有哪些待办任务
        1. 服务生效应：未完成的任务总是更容易被记住
      3. 思考[此时此刻，我的时间最好用来做什么]
      4. 可以处理一下邮件信息，或者回应一下他人的聊天

  6. 区分自己的时间类型

    1. 可支配时间和不可支配时间
      1. 可支配时间：可以自己决定这段时间里面做什么。
      2. 不可支配时间：不可以决定自己这段时间里面做什么。比如说开会。
        1. 不可支配时间必须被记录在日程表中，避免忘记或者与其他任务发生冲突
    2. 保证自己的可支配时间和不可支配时间平衡
      1. 不平衡的表现：任务进度没有达到预期
      2. 不平衡的处理方式：缩小自己的责任范围，或者，增加平且保护自己的可支配时间

  7. 专注当下的事情

    1. 不进行多任务的切换
    2. 采用单核和全景来对时间进行划分，并在对应的时间段执行对应的任务
    3. 在单核时间段关掉所有的可能的通知，无论是电子邮件还是短信还是聊天软件
    4. 引导可能突发的可能导致当前任务中断的任务到其他时间段进行解决
    5. 将重要的任务排在紧急任务之前，当下便能够处理最重的事情，如此避免开时间上的压力

  8. 提高工作效率

    1. 每周进行至少150分钟的运动，如此白天会更加的清醒，不容易出现注意力的溃散
    2. 膳食均衡能够给大脑提供更加稳定的能量输出，水果和蔬菜能够让人更加的具有创造力
      1. 一天到晚都吃糖果和甜点会导致或者喝汽水会导致血糖水平的忽高忽低
      2. 如果午餐摄入一些高热量的食物则会导致下午昏昏欲睡
      3. 一周吃五次左右的蔬菜和水果能够让生产力提高20%左右
      4. 在进行需要大量消耗脑力的工作的时候可以摄入少量的糖分来给大脑提供能量，比如说一颗糖
      5. 在大脑处于低能量的时候不要做重要的决定
    3. 充足的睡眠能够让人完成对记忆的编码，整理白天接受到的信息和删除无用的内容
      1. 短期睡眠不足只会影响另一天的效率，但是长期睡眠不足会降低一个人的智力
      2. 卧室或者床头采用暖色系的黄光能够让人更加的容易入睡
      3. 睡前90分钟左右进行热水沐浴，睡前10分钟左右进行热水足浴能够让人更加的容易入睡
      4. 一天的睡眠时间需要在9个小时左右，最好在浅睡眠时期醒来
      5. 如果在日间进行睡眠补觉，时间不能够超过半个小时
    4. 人的大脑不能够长时间的专注于某一个任务
      1. 需要在一定的时间之后强制自己进行休息
      2. 可以在休息之后切换到另一种类型的任务上

  9. 轮流使用发散模式和专注模式来帮助思考

    1. 专注模式当人一心一意做某件事情的时候会出现
      1. 人的大脑中只有局部的神经区域处于激活状态
      2. 人的思路会像一个弹球，在这个区域内快速弹射，然后找到一个解决问题的路径
      3. 需要人掌握了相关的知识，或者，具有相关的经验
      4. 激活的区域内知识之间的联系会被加强
    2. 发散模式当人思维开始发散或者注意力开始溃散的时候会出现
      1. 人的大脑的不同区域的神经联系会被激活
      2. 解决问题或者学习的效率会开始下降
      3. 人的创造力开始上升，不同的领域的知识可能被链接来给出解决方案

  10. 对自己所需要学习的知识进行组块化

    1. 组块是存储在人的大脑中的小而紧凑的信息包
      1. 小：不是繁多巨大
      2. 紧凑：信息的关联性强，没有多余的信息
    2. 组块化遵循三个步骤
      1. 对想要组块化的信息全神贯注，也就是启用专注模式，不要不断的切换任务
        1. 人的大脑在高度兴奋的情况之下会导致治时间的联系能力下降
        2. 在生气，紧张或者害怕的时候不要开始学习，而是先试着冷静自己的大脑
      2. 对建立组块的对象有基本的了解
        1. 可以使用专注模式来理解要被建立成组块的对象
        2. 可以使用发散模式来思考该知识点和其他知识点之间的关系
        3. 当一个人能够使用这个组块来解决对应问题的时候，才算做是掌握了这个知识点
          1. 可以思考这个组块会对应什么问题
          2. 对于练习习题不要先看答案，也不要认为能够看懂答案就是掌握了知识点
      3. 获取组块的背景知识
        1. 了解什么时候要使用这个组块，什么时候不使用这个组块
        2. 让组块能够融入整个知识框架，而不是独立于框架之外
    3. 进行不断的反复的练习来强化自己所掌握的组块，和建立不同组块之间的关联

  11. 采用费曼技巧来提供自己的学习新知识的效率

    ​

    1. 明确自己要学习的目标
      1. 将自己要学习的主体或者概念写在这张白纸上
      2. 要有清晰的概念，而不是模糊的印象
    2. 模拟将学习目标的内容交给一个不会的人
      1. 假设自己是老师在教导学生，要怎么教导相关的内容
      2. 教导相关的概念
      3. 教导概念和概念之间的联系
    3. 如果在教导的过程中卡壳，则回顾自己学习的内容
      1. 了解自己忽视的内容是什么
      2. 起到了反馈的作用
    4. 回顾后重新用简化的非行业术语来进行表达

  12. 进行刻意的练习来掌握新知识和巩固自己掌握的知识

    1. 进行刻意学习的两个标准
      1. 是合理发展的领域并且有着完成成熟的评价标准
        1. 能客观的测量效绩
        2. 具有竞争性
        3. 存在着已经被实践了并且具备科学性的训练方法
        4. 最杰出的从业者已经达到了一定程度的表现水平并且有着一套行之有效的训练方法
      2. 需要有一个能够给自己布置训练作业并且给予及时反馈的优秀的导师
        1. 寻求该领域的成功者的帮助，并且请求他们为自己制定一套训练方法
        2. 尝试着对成功者进行模仿来提升自己
    2. 刻意学习有四个原则
      1. 明确高绩效的目标，也就是哪些方面需要提升
      2. 尽可能的去寻找这个领域的专家，成功者，高手或者经典的书籍，以此来和自己进行不断的对比
      3. 研究那些杰出的人，去研究他们杰出背后可能的原因是什么
      4. 不断的投入时间和精力来进行训练
    3. 刻意练习发生在舒适区之外
      1. 舒适区：已经完全掌握的区域，没有学习难度的知识或者习以为常的事务。
      2. 学习区：还没有完全掌握的区域，对你来说有一定挑战，感到不适，但不至于太难受。
      3. 恐慌区：超出自己能力范围太多的事务或知识，心理感觉会严重不适，可能导致崩溃以致放弃学习。
